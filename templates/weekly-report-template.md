## 📖 생성 가이드

### YAML Front Matter 필드 설명
- `week_number`: 주차 (연도 기준)
- `year`: 연도
- `start_date`: 주간보고 시작일 (월요일)
- `end_date`: 주간보고 종료일 (일요일)
- `author`: 작성자
- `generated_at`: 자동 생성 일시
- `source_files`: 참조한 일일 업무일지 파일 목록

### n8n 자동화 처리 로직
1. **일일 업무일지 수집**: 지정된 기간의 모든 일일 업무일지 읽기
2. **카테고리별 분류**:
   - Redmine → 일감번호 추출
   - 자동화 → Server-i/WebKeeper/PV 등 세부 분류
   - 기타 → 인프라/문서화/회의/학습 분류
3. **상태별 정리**:
   - "오늘 완료한 업무" → 완료
   - "진행 중인 업무" → 진행중
   - "내일 할 일" → 다음 주 계획
4. **통계 생성**: 소요시간 합산, 업무 건수 집계
5. **AI 요약**: 주요 성과 및 특이사항 자동 요약
